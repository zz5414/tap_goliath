# Goliath Drift (골리앗 드리프트)

> **단 하나의 버튼**으로 플레이하는 탑다운 오토 슈터. Vampire Survivors 형식의 캐주얼 서바이벌 게임.

🎮 **플레이하기:** [https://tap-goliath.vercel.app/](https://tap-goliath.vercel.app/)

---

## 소개

본체(Goliath)는 항상 일정한 속도로 한 방향을 향해 자동으로 이동합니다. 화면 어디든 **탭(또는 클릭/스페이스바) 한 번**을 누르면 이동 방향이 바뀝니다. 사격은 자동입니다.

- **의도된 불편한 조작감**: 어느 방향으로 튀어나갈지 완전히 통제할 수 없습니다. 그 안에서 *최선의 타이밍*을 찾는 게임입니다.
- **자동 사격 + 수동 회피**: 가만히 있을수록 조준이 정확해지지만, 적에게 둘러싸이기 전에 누를 수밖에 없는 압력이 생깁니다.
- **레벨업 빌드업**: 처치한 적이 떨어뜨리는 경험치를 모아 레벨업하고, 카드 3장 중 하나를 골라 골리앗을 강화합니다.
- **모바일 친화**: 터치 한 번으로 플레이. 가로/세로 어떤 화면에서도 작동합니다.

## 조작

| 디바이스 | 입력 | 동작 |
|---|---|---|
| 모바일 | 화면 탭 | 방향 전환 |
| 데스크톱 | 마우스 좌클릭 | 방향 전환 |
| 데스크톱 | 스페이스바 | 방향 전환 |

레벨업 카드는 **더블탭으로 확정**됩니다.

## 기술 스택

- **TypeScript** (strict)
- **Vite** (빌드 / HMR)
- **HTML5 Canvas 2D** (런타임 의존성 0개)
- **Vercel** (배포)

## 로컬 실행

```bash
bun install        # 또는 npm install
bun run dev        # 개발 서버 (HMR)
bun run build      # 프로덕션 빌드
bun run preview    # 빌드 결과 미리보기
```

## 프로젝트 구조

```
src/
├── main.ts          # 엔트리포인트
├── game/            # 게임 루프, 카메라, 입력
├── entities/        # 본체/포탑/적/총알/경험치 구슬
├── systems/         # 방향 선택, 스폰, 충돌, 경험치 등 시스템
├── render/          # 렌더러, HUD
├── ui/              # 시작 메뉴, 레벨업 메뉴, 게임오버 화면
├── config/          # 밸런스 수치 (튜닝은 여기서만)
└── util/            # 수학/랜덤 헬퍼
```
